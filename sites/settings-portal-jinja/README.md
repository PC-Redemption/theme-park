# Settings Portal Jinja

Settings Portal Jinja is a Jinja-first starter for admin, health, and configuration workflows.

Its job is to prove that the shared framework also supports table-heavy settings portals, user administration, event logs, and health-review pages.

## Local Structure

```text
sites/settings-portal-jinja/
  layouts/
  scaffolds/
  templates/
  app_config/
  routes/
  static/
```

## Runtime

- templating model: Jinja
- expected backend fit: FastAPI, Flask, or another Jinja-capable runtime

## Preview

This starter follows the repo `.venv` plus small preview script policy.
