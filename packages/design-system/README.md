# Design System

This package contains shared framework assets used across starter sites.

## Scope

Shared code belongs here when it is reusable across multiple starters without site-specific naming or behavior changes.

Examples:

- theme tokens
- shared styles
- shared scripts
- reusable components
- integration helpers

## Boundaries

- do not place starter-specific templates or content here
- do not import private product logic here
- promote code here only after confirming it is broadly reusable

## Planned Structure

```text
packages/design-system/
  tokens/
  styles/
  scripts/
  components/
  integration/
```
